- TikZbricks ( [CTAN](https://www.ctan.org/pkg/tikzbricks), [github](https://github.com/samcarter/TikZbricks) )
- Jigsaw ( [CTAN](https://www.ctan.org/pkg/jigsaw), [github](https://github.com/samcarter/jigsaw) )
